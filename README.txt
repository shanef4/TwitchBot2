Command List:

!points - Display user's points.
!save - Saves current state of points. (only usable by the account cpt_skip)
!gamble X - Gambles X points

!2x2 - Displays 2x2 according to emotes set in settings.
[emote1][emote2]
[emote3][emote4]

!loyal - Displays user's loyalty points, add them to the regular list if they have the required amount or more.

The Edit A User's Points button will allow you to change a user's current points. The points it changes are the points the user can gamble.
